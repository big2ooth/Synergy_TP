 *brief description*: 
  The main objective of this assignment is to build a small Python program inside the task_2 folder using functions, lists, dictionaries, file I/O, exception handling, and type hints.

# What the program does 
- Reads student data from submissions.csv
- Calculates total, submitted, and missing students
- Finds average score, highest and lowest scorer
- Calculates domain-wise average scores
- Finds students scoring below 5
- Writes all results to output/summary.json
